Run Mode Guidance

For live API predictions, start the backend service using run_backend_service.bat (or python notebook_backend_service.py). The dashboard connects to http://127.0.0.1:5000/api and renders real-time analytics. If the backend is unavailable, a fallback simulation mode remains active using dataset-driven trend analytics.
